# Alex Virendra Sakha Portfolio

Version 0.2 of a fast, semantic engineering portfolio and journal.

## Architecture

- `index.html` contains the page structure and section placeholders.
- `assets/css/styles.css` contains the design tokens, layout rules, and responsive behavior.
- `assets/js/main.js` contains small progressive enhancements for navigation, profile-photo readiness, and the SGPA SVG chart.

The site is intentionally framework-free and GitHub Pages-compatible. Future features should land as small patches to a section, asset file, or new page rather than requiring a rebuild of the foundation.

## Content Notes

The current patch includes only resume facts available in the repository or provided during the request. Replace `Resume detail required` entries with exact resume values before treating the site as content-complete.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.
